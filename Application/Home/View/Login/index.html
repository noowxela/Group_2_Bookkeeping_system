About 小歆记账

小歆记账 is an open source, light weight, portability cloud accounting apps, the system is mainly using ThinkPHP, Accounting data is store in MySql, it can be build on any PHP+MySql.

This Accounting apps can freely create any type of income/expenses and account, personalized query function will easily let user to find the history transaction, for long term usage, it can show statistics and graphs for personal viewing, to show the overall income and expenditure effectively.
